# 🚀Project Overview

This project analyses operational performance in the iGaming sector using player activity and financial transaction data.
The goal is to transform raw operational data into meaningful business insights through structured analytics and visual dashboards.

The analysis focuses on:

- Player acquisition and activity
- Deposit behaviour
- Turnover and winnings relationship
- Bonus allocation across markets
- First time depositor behaviour
- Country level performance differences
  
---

# 🎯 Problem Statement

iGaming operators manage large volumes of transactional data across multiple markets. Without structured analytics, it becomes difficult to answer key business questions.

Examples of operational questions include:

- Which countries generate the most active players
- Which markets produce the highest deposit value
- Whether bonus spending improves deposit behaviour
- Which markets show high winnings relative to turnover
- Where first time depositor rates are strongest

# 🔎 Data Analysis Process

The project followed a structured analytics workflow.

### Step 1 Data Collection

Gaming operational data was imported from CSV and Excel datasets.

### Step 2 Data Cleaning

- Standardised numeric values
- Checked for missing values
- Verified country level consistency

### Step 3 Feature Creation

Derived performance indicators including:

- Winnings to Turnover ratio
- Winnings to Deposit ratio
- Average bonus by market
- Active player segmentation

### Step 4 Exploratory Data Analysis

Country level comparisons were performed across:

- player activity
- deposit behaviour
- bonus allocation
- turnover efficiency

### Step 5 Dashboard Development

Interactive dashboards were built to visualise:

- player distribution by country
- financial performance metrics
- bonus allocation across markets
- deposit conversion indicators

---

# 💼 Executive Summary

Key observations include:

- Certain markets generate significantly higher active casino players.
- Turnover to winnings ratios vary across countries.
- Bonus distribution is concentrated in a small number of markets.
- First time depositor rates differ strongly between countries.

---

# 💡 Recommendations

Based on the analysis, several operational recommendations emerge.

- Optimise bonus allocation in markets with low deposit conversion.
- Focus marketing investment on high performing markets.
- Improve the player conversion funnel from registration to deposit.
- Monitor winnings to turnover ratios for potential risk signals.

---

# 🧾 Conclusion

This project demonstrates how gaming operational data can be transformed into actionable business insights.

By integrating player activity, financial metrics, and promotional spending into a unified dashboard, operators gain improved visibility into platform performance.
